Tailwind CSS 实用技巧

2025-08-31 作者：陈优嘉 245 阅读

TailwindCSS 凭借原子化类名大幅提升开发效率，但掌握一些实用技巧，能让开发更顺畅。

一、自定义配置简化开发

不用重复写复杂类名，在 tailwind.config.js 中自定义主题。比如项目常用特定颜色 “品牌蓝”，可在 theme.colors 里添加：brandBlue: '#165DFF'，之后直接用 text-brandBlue“bg-brandBlue” 调用，既统一风格又节省时间。

二、响应式设计快速适配

借助 Tailwind 预设的响应式前缀（sm:、md:、lg: 等），无需写媒体查询。要实现 “移动端文字居中，桌面端文字左对齐”，只需写

三、解决类名冲突方便

当第三方组件样式与自定义样式冲突时，用 @apply 整合类名并提升优先级。例如：​

.custom-btn {​
@apply bg-blue-500 text-white py-2 px-4 !important;​
}
